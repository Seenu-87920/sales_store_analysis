🛒 Sales Data Cleaning & Business Analysis using SQL Server
📌 Project Overview

This project demonstrates an end-to-end SQL data analytics workflow using Microsoft SQL Server.

The project starts with importing raw sales data from a CSV file, followed by comprehensive data cleaning techniques and finally solving real-world business problems using SQL queries.

It is designed to showcase practical SQL skills required for Data Analyst and Business Analyst roles.
